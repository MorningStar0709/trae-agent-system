### Overview

Welcome to the **Trae Skills** repository! This project contains a collection of custom skills and agents designed to enhance the capabilities of the [Trae IDE](https://www.trae.ai/). These skills are aimed at automating repetitive tasks, establishing development rules, and streamlining your coding workflow.

### Available Skills

- **agent-blueprint-architect**: Assists in creating, designing, and optimizing concrete Trae agent configurations.
- **creating-trae-rules**: Helps define and organize Trae project rules (`.trae/rules`) and team conventions.
- **skill-creator**: A dedicated skill for scaffolding, reviewing, and iterating on new Trae skills.
- **skill-stability-review**: Audits and reviews Trae skills packages to ensure Windows/Trae adaptation, execution stability, and correct triggers.

### Getting Started

To use these skills in your Trae IDE:
1. Clone this repository to your local workspace.
2. Follow the specific instructions provided in each skill's `SKILL.md` file.

<h2 id="中文">中文</h2>

### 项目简介

欢迎来到 **Trae Skills** 仓库！本项目包含一系列为 [Trae IDE](https://www.trae.ai/) 量身定制的自定义技能（Skills）和智能体（Agents）。这些技能旨在自动化重复性任务、规范开发规则，并优化你的编程工作流。

### 包含的技能

- **agent-blueprint-architect**：帮助创建、设计和优化具体的 Trae 智能体（Agent）配置。
- **creating-trae-rules**：协助定义和组织 Trae 项目规则（`.trae/rules`）及团队规范。
- **skill-creator**：专门用于脚手架搭建、审查和迭代新 Trae 技能的工具。
- **skill-stability-review**：审计和审查 Trae 技能包，确保 Windows/Trae 适配性、执行稳定性和触发边界的正确性。

### 快速开始

在 Trae IDE 中使用这些技能：
1. 将此仓库克隆到本地工作区。
2. 遵循每个技能目录下 `SKILL.md` 文件中提供的具体使用说明。
